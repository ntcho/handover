<h2 id="features"> 기능 설명 (Key Features) ✨</h2>

Handover만의 **3가지 핵심 기능**과 함께 더 편리하고 똑똑한 인수인계를 진행해보세요!
 - **`3단계 문서 구성`**(서랍 🗄️, 문서 📰, 카드 📇)으로 **쉽게 읽고, 작성하는** 인수인계 문서
 - **`그룹 생성`**, **`북마크`**, **`댓글`** 기능들로 **함께 공유하는** 인수인계 문서
 - **`보안 관리자 기능`** 으로 보안위반으로부터 **안전한** 인수인계 문서

<h3> 3단계 문서 구성 📋</h3>

<p align="center">
 <img src='http://drive.google.com/uc?export=view&id=1uoMROdL5DH_qtouCaIKYWde_3wmi9pKq' width="800" height="500"/>
</p>

Handover 내의 모든 문서는 **서랍 🗄️, 문서 📰, 카드 📇** 의 3단계 구성을 갖습니다.
- **서랍** 🗄️: 하나의 부서 또는 보직에 연관된 업무 파일들의 집합
- **문서** 📰: 하나의 업무를 구성하는 카드들의 집합
- **카드** 📇: 작은 업무를 수행하는 방법을 설명한 짧은 문서

<p align="center"> 
 <img src='http://drive.google.com/uc?export=view&id=1A4XxkjSMLXGl4dtebVH6KG0wfLqCMLMT' width="400" height="400" />
</p>

3단계로 구성된 문서에 더해, 미리 제공된 **카드 템플릿**들을 이용해 더 읽기 쉽고, 체계화된 문서를 작성해보세요!

<h3> 그룹, 북마크, 댓글 기능을 통한 문서 공유 서비스 🤝 </h3>

<p align="center"> 
 <img src='http://drive.google.com/uc?export=view&id=1A4XxkjSMLXGl4dtebVH6KG0wfLqCMLMT' width="400" height="400" />
</p>

- 같은 보직, 같은 소속의 사용자들과 **그룹**을 형성하여 인수인계 문서를 공유하고, **댓글**을 통해 의견을 나누어보세요!
- 마음에 드는 문서가 있다면, **북마크** 기능을 통해 언제든지 문서를 다시 열람할 수 있습니다.

<h3> 보안 관리자 등록을 통한 기밀 유출 방지 🔐 </h3>

<p align="center"> 
 <img src='http://drive.google.com/uc?export=view&id=1A4XxkjSMLXGl4dtebVH6KG0wfLqCMLMT' width="400" height="400" />
</p>

- 그룹 별로 **보안 관리자**를 등록해 그룹 내 문서에서 보안 사항이 위반되고 있는지 확인하고, 관리할 수 있습니다.

<h2 id="prerequisites"> 컴퓨터 구성 / 필수 조건 안내 (Prerequisites)</h2>

* ECMAScript 6 지원 브라우저 사용
* 권장: Google Chrome 버젼 77 이상

<h2 id="stacks"> 기술 스택 (Key Tech Stacks)</h2>

### Back-end
* `RESTful`한 백엔드 구축
    - [`nodejs 14.17.6`](https://nodejs.org/en/)
    - [`expressjs`](https://expressjs.com/)
    - [`Docker`](https://www.docker.com/)
* `NoSQL` 데이터베이스 사용 
    - [`MongoDB`](https://www.mongodb.com/)
* `Algolia`를 사용한 검색
    - [`Algolia Search`](https://www.algolia.com/)
 
### Front-end
* `React`를 사용한 프론트엔드 구축
    - [`react.js`](https://reactjs.org/)
* `Material UI`를 사용한 디자인
    - [`MaterialUI`](https://mui.com/)
